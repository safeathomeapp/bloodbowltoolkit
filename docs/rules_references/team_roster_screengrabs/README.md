# Team Roster Screengrabs

Upload rulebook or authoritative roster screenshots for team-template entry here.

## Purpose

These images are the source material for creating and validating roster template seed data in `modules/team-creator/`.

## What To Upload

- one screenshot per team roster page where possible
- cropped images that clearly show:
  - team name
  - league
  - special rules
  - reroll cost
  - apothecary availability
  - all position rows
  - primary and secondary categories

## Naming Recommendation

Use simple stable filenames, for example:

- `amazon_roster_rulebook.png`
- `orc_roster_rulebook.png`
- `human_roster_rulebook.png`
- `skaven_roster_rulebook.png`

## Practical Rule

Do not overwrite earlier files if a new crop or clearer image is added.

Prefer adding a clearer replacement with a version suffix, for example:

- `orc_roster_rulebook_v2.png`
