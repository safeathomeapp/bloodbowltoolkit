# 2026-05-14 Edit Team Label Colour Names

## Summary

This pass changes the edit-mode top-level team toggle labels from `Team A` / `Team B` to `Blue` / `Red`.

## Why This Changed

- The rest of the calculator already leans on color-coded team identity.
- `Blue` / `Red` is clearer in the current UI than mixing color language with internal `A` / `B` team identifiers.
- This reduces friction when moving between the edit controls and the board-header active-side toggle.
